# Kehoachtuan - PGD Task Tracker (GitHub Pages)

## Fix package
Bộ file này dùng để thay thế trực tiếp các file trong repo GitHub Pages:
- index.html
- app.js
- config.js
- styles.css
- supabase.sql

## Cách cập nhật
1) Vào repo GitHub của bạn -> upload/replace các file trên.
2) Đợi GitHub Pages cập nhật (hoặc Ctrl+F5 để xoá cache).
3) Nếu bấm "+ Thêm việc" không hoạt động: mở F12 -> Console để xem lỗi.
