# Kehoachtuan (GitHub Pages)

## Cách cập nhật (fix lỗi không thêm được công việc)
1. Vào repo: https://github.com/anhvt777/Kehoachtuan
2. Upload 4 file này vào ROOT (cùng cấp index.html), chọn Replace nếu trùng:
   - index.html
   - styles.css
   - config.js
   - app.js
3. Mở lại website: https://anhvt777.github.io/Kehoachtuan/
4. Nhấn Ctrl+F5 hoặc mở tab ẩn danh để tránh cache.

## Ghi chú
- Dữ liệu mặc định lưu localStorage trên từng thiết bị.
- Nút 'Xuất báo cáo tuần' xuất XLSX nếu CDN xlsx chạy được; nếu bị chặn sẽ xuất CSV.
